# Agent-musicman



一，语言模型
1,通过对InternLM2.5-7b-instruct微调实现其对自我认知的理解
2，蒸馏闭源模型并从github上获取有关古典音乐数据集完成指令微调数据集获取
3，微调大语言模型

二，对于复杂领域古典音乐数据
1，采用llama-index的索引方式，完成对于复杂古典音乐特殊典籍的理解
2，除了语言模态的RAG，考虑多模态RAG方式

三，采用agent-lego
采用agent相关技术，能够完成对于相关复杂古典音乐典籍网站的api查询以及相关利用方式

四，基于多模态大模型
基于多模态大模型完成对于古典音乐谱的赏析，先采用能力较强的闭源多模态大模型进行标注，再采用对多模态大模型微调完成相关的任务。
